The content of the cards uses AsciiDoc notation.

== AsciiDoc notation

AsciiDoc is a lightweight markup language primarily designed for writing technical documentation. It has a rich notation that supports, for example, headings, tables, lists, admonitions, footnotes, and links.

For a reference guide to AsciiDoc notation, see https://docs.asciidoctor.org/asciidoc/latest/syntax-quick-reference/[AsciiDoc Quick Reference].
