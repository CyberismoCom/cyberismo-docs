The Cyberismo solution architecture is based on the following principles:

* The Cyberismo tool is installed on users' computers.
* The Cyberismo tool may also be run in the CI/CD pipeline.

The tool supports the following interaction modes:

* A command-line interface
* A local web browser interface (_Cyberismo App_)
* Generating a static website from the Cyberismo project content
